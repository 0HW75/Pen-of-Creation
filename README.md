# Pen-of-Creation

AI小说编辑器 - 一个强大的AI辅助小说创作工具

## 项目介绍

Pen-of-Creation是一个基于AI的小说编辑器，旨在帮助作家更高效地创作小说。

## 功能特性

- 📝 智能写作工坊：AI辅助写作，提供灵感和建议
- 📊 作品分析：分析作品结构和风格
- 🗺️ 创作导航：可视化故事结构
- 🧠 创作智库：管理角色、地点、物品等设定
- 📄 作品导出：支持多种格式导出
- 🔄 实时一致性维护：确保故事一致性

## 技术栈

### 前端
- React
- Vite
- JavaScript

### 后端
- Python
- Flask
- SQLite

## 快速开始

### 前端
```bash
cd frontend
npm install
npm run dev
```

### 后端
```bash
cd backend
pip install -r requirements.txt
python run.py
```
